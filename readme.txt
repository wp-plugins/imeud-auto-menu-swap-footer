Scrolling auto swap nav tag to the bottom.

== Description ==

As scrolling (not the bottom), the nav fade out, and as scroll to the bottom to see the show again. * Effective with Theme with nav tag.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `imeud-auto-menu-swap-footer` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
